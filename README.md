# AI Trust Analytics

An end-to-end product analytics case study identifying the psychological drivers of user trust in artificial intelligence.

## Research Question

Which psychological and demographic factors most strongly influence users' trust in artificial intelligence, and how can these findings inform product design decisions?

## Tools Used

- Excel
- SQL / SQLite
- Python: Pandas, Statsmodels
- Tableau

## Project Workflow

1. Defined research question and project scope
2. Documented dataset and variables in Excel
3. Explored and validated data using SQL
4. Cleaned data and engineered construct scores in Python
5. Ran descriptive statistics, correlation analysis, and regression analysis
6. Built a Tableau dashboard
7. Translated findings into product recommendations

## Key Findings

- Competence was the strongest independent predictor of Trust in AI.
- Warmth had the strongest overall correlation with Trust in AI.
- Robustness and Technology Trust significantly contributed to trust.
- The regression model explained approximately 61% of variation in Trust in AI.

## Product Recommendations

- Improve AI competence
- Increase warmth in AI responses
- Improve system robustness
- Strengthen privacy communication
- Build user confidence through education
